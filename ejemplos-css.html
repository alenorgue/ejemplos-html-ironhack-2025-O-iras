Ejemplos CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje que permite dar estilo a los elementos HTML, como los colores, tamaños y distribución.

El atributo style

Permite definir estilos directamente en una etiqueta HTML.

Ejemplo: <p style="color: blue;">Texto azul</p>

Propiedades CSS comunes

  • color: Cambia el color del texto.
  • background-color: Cambia el color de fondo.
  • font-size: Define el tamaño de la letra.
  • font-family: Define el tipo de letra.
  • text-align: Alinea el texto (izquierda, centro, derecha).
  • text-decoration: Agrega subrayado, tachado, etc.

La etiqueta <style>

Permite definir estilos en un solo lugar dentro del <head> del documento.

Selectores CSS básicos

Tag selector

Aplica estilos a todas las etiquetas de un tipo. Ejemplo:

p { color: green; }

Class selector

Aplica estilos a elementos con una clase específica. Ejemplo:

.info { background-color: lightgray; }

El atributo class

Permite agrupar elementos para aplicarles estilos con CSS. Ejemplo:

<div class="info">Contenido aquí</div>

¿Cuándo usar unos y cuándo usar otros?

  • El tag selector es útil cuando queremos aplicar estilos a todas las etiquetas de un tipo.
  • El class selector es mejor cuando solo queremos estilizar ciertos elementos.
